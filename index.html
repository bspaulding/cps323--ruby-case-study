Writing Scripts

use gedit, because its light and has ruby syntax highlighting.

you can run stuff like this:

ruby script.rb

or declare the interpreter type within the script:

#! /usr/bin/ruby
puts "Hello World!"

and then (making the file executable) run it like an executable:

./script.rb
